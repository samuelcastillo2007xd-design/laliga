# Liga de Fútbol en C++

## Autor
Samuel mauricio castillo españa

## Descripción
Sistema en consola que gestiona una liga de fútbol con equipos, partidos, tabla de posiciones e historial de jornadas.

## Cómo compilar
g++ src/main.cpp -o liga

## Cómo ejecutar
./liga

## Archivos
- config.txt: configuración de liga y equipos
- partidos.txt: registro de partidos
- fechas.txt: historial por jornadas
- tabla.txt: tabla exportada
